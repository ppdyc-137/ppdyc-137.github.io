emby,linux的一些后台指令

Posted on 2020-10-12 | In 日志

Emby:
路径：/usr/local/media/
端口：8096
好像挺简单

防火墙开启端口：
打开单个端口：firewall-cmd –zone=public –add-port=80/tcp –permanent
重启防火墙：systemctl restart firewalld.service

linux后台运行相关指令：
对于已经在前台运行的程序：
# Ctrl+Z：暂停当前前台程序
# bg + 进程号：将该进程放置后台运行
# jobs：列出当前后台进程
# fg + 进程号：将该进程放回前台运行
# kill：终止进程
在程序启动前：
# command &：后台运行command命令
但是当程序在后台运行有输出时，还是会输出到前台：

command >out.file 2>&1 & ：后台运行command并将输出重定向到out.file里

这样程序能在后台运行且不影响前台，但是关闭终端后会终止运行：
#nohup command & ：后台运行command且关闭终端后仍然运行
其他：
#ps -ef grep name: 列出带有name的进程
